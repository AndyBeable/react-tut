## JSX

Special dialect of JS (its not html)
Browsers don’t understand JSX. We run tools (Babel) which convert it in to normal JS.
Allows us to write out many function calls in a format that is much easier to understand.
Very similar in form and function to html

## JSX vs HTML

Adding custom styling to an element use different syntax
Adding a class to element use different syntax
JSX can reference JS variables
