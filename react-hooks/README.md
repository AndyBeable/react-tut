## Hooks System - are a way to write reusable code instead of more classic techniques like oop, inheritance

Gives function components additional functionality as they cant make use of state or lifecycle methods.

Hooks gives us functions such as:

Primitive Hooks

- UseState
- UseEffect
- useRef
- UseContext
- Use Reducer
- UseCallback
- UseMemo
- UseImperativeHandle
- UselayoutEffect
- UseDebugValue

You can also create your own Custom Hooks.
